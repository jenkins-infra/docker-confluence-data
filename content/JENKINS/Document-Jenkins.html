Jenkins Documentation

Jenkins.io includes a Documentation section to provide product documentation beyond what is provided by the plugin pages specific to each plugin. This documentation is more focused on higher level use cases and concepts in Jenkins rather than just how to configure a feature or plugin.

Contributing new documentation is done via the Jenkins.io Github repository and should be written in AsciiDoc.
