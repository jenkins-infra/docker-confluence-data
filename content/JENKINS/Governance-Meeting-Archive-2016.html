October 12 meeting

WHEN October 12 11:00 Pacific.

  • Recap last meeting's actions
  • LTS status check (Unknown User (olivergondza))
  • Bumping Jenkins core to Remoting 3 (Unknown User (oleg_nenashev))
    • Remoting 3 offers a new JNLP4 protocol (Adds Java NIO compared to JNLP3 + stability fixes)
    • Remoting 3 is formally incompatible: Java 6 support drop from slaves, also JnlpServerHandshake rework: https://github.com/jenkinsci/jenkins/pull/2492
    • ~stephenc checked known OSS and closed-source implementations, no real compatibility issues
    • Question for voting: Do we want to go forward?
  • Out-of-order Jenkins Weekly release in order to get better soak testing for 2.19.3 (Unknown User (oleg_nenashev))
  • Question for voting: Do we want to release it if kohsuke is available?
